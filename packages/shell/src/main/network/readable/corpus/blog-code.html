Debouncing in TypeScript

A debounce delays a function until activity settles. Here is a small, dependency-free implementation you can drop into any project today.

This paragraph carries enough genuine prose for Mozilla Readability to score the article body above the surrounding navigation and sidebar chrome, which is the whole point of the readable-extraction pass.

const debounce = (fn, ms) => { let t; return (...a) => { clearTimeout(t); t = setTimeout(() => fn(...a), ms); }; };

The closure keeps the timer handle private, and each call resets it.
